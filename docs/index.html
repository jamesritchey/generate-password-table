Ritchey Generate Password Table i1 v1

This tool generates a table that can be used to create deterministic passwords for services/devices. The table is generated from a master password. So long as the user has a unique master password, they will have a unique table. An offset number can be used to generate alternate tables for the same service/device.
Generate Table
The table will contain sets of password acceptable characters associated with each letter of the alphabet. This table can be used to create passwords for any service or device by combining the sets of characters for each letter of the service/device name.

Match the letters in the name of the service/device to the table, and note the values. Trim the result to desired password length.

Notes
  • The password length input can be used to trim a password to a specific length, but won't pad a shorter one to grow it. If the service/device name isn't long enough, this means the password could be shorter than the specified length.
  • All table values contain a lowercase letter, uppercase letter, number, and special character. They also always start with a lowercase letter. Using this approach reduced the complexity of developing this tool. However, as a result, it rules out many possible passwords that meet generally acceptable password criteria.
  • To improve security, consider replacing 'combinations_random_1.js' with a CUSTOM copy in which the values are randomly sorted.
